靠 BDD + DDD 做到 AI 100% 全自動化後端開發
水球潘

我只講一件事，要做到 100% 後端 AI 全自動化開發，該做哪些事？ 我講的是 100% 全自動化，不是 80% 也不是 90% 也不是 99.9%，也不是 99.99999%。 我說的是 100%。 所謂的 100% 的意思就是你只要提交完規格之後，就可以完全不 code review，可以去玩遊戲的那種 100%。 從 70% 一路到 100% 全自動化，你知道最重要的是什麼嗎？ 是 Prompt? 不，怎麼可能是 Prompt，Prompt 根本無關緊要。其實答案是 BDD 和 DDD 這兩大重要實踐的極致落地。 你各位對 DDD 有關注的人全部都賭對了！ 要做到 100% 全自動化開發，如果只會寫規格，寫可執行規格，寫各種 Given When Then，但是卻沒有 DDD 的本事的話，是不可能做到的！ 如果有 DDD 的思維卻沒有 BDD 的可執行性的話，那也是無法做到全自動化開發的。 一切都要做好做滿，雖然成本很大，但就能得到 100% AI 全自動化開發的甜美果實。
